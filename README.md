# Orçamento - PDF
Sistema em Python para geração automática de orçamentos. Os usuários fornecem detalhes do projeto, e o sistema calcula e gera um PDF com o orçamento, simplificando o processo.
## Tecnologias Usadas
* [Python](https://www.python.org/)
